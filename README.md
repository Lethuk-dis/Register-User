Installation & Running                                     

Install Netbeans IDE(latest version) 

Install JDK 17 or higher 

Install MySQL server, https://dev.mysql.com/downloads/ 

Install MySQL Connect/J, https://dev.mysql.com/downloads/connector/j/ 

Create a Database(user_registration_db), with table users 

Open the project in NetBeans 

Update database credentials in the code (DB URL, username, password) 

Build the project (Clean and Build) to generate the JAR file 

Run the main class
